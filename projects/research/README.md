# Research Projects

Research projects are the most open-ended track in the repository. They are intended for
publication-style final work, original methodological exploration, or carefully bounded companion
essays that connect canonical ML to category theory or Unity Theory.

## Track purpose
Use this track when the project is expected to make an original argument, prototype a novel method,
or propose a disciplined interpretive synthesis. Canonical ML competence remains mandatory; the
research layer is additive, not a substitute for standard reasoning.

## Submission expectations
Each submission should include:

- a proposal memo with question, novelty claim, risks, and planned evidence;
- recurring advisor-style check-ins with written decisions after each milestone;
- a final paper suitable for workshop-style or internal lab review;
- reproducible appendices, code, diagrams, or notebooks when the project has an empirical or
  algorithmic component; and
- a scope statement marking which claims are standard, interpretive, conjectural, or speculative.

## IP and authorship policy
- Students should treat these projects as potentially publishable and keep records of major
  contributions, collaborators, and source materials.
- Any project using outside data, model outputs, or generated text must document provenance and
  licensing constraints.
- Unity Theory work must state clearly whether it is commentary, a correspondence proposal, or a
  speculative research extension.

## Prompt set
- [`unity-theory-synthesis-essay/`](./unity-theory-synthesis-essay/README.md): write a disciplined
  Unity Theory essay on a canonical ML topic.
- [`compositional-algorithm-design/`](./compositional-algorithm-design/README.md): design and
  justify a novel algorithm or training procedure using compositional principles.
- [`cross-domain-transfer-analysis/`](./cross-domain-transfer-analysis/README.md): analyze transfer
  across domains through a structural or categorical lens.

## Evaluation
Use [`rubric.md`](./rubric.md) for track-level grading. This rubric separates `technical rigor`
from `speculative quality` so ambitious work is rewarded for discipline rather than hype.
