# Module 13: Graph Learning

## Purpose

Study learning problems where the data carry an explicit relational structure.
The module connects graph representations, Laplacian-based spectral methods, message passing, graph neural networks, and graph transformers.

## Prerequisites

- `00-math-toolkit/01-linear-algebra`
- `06-neural-networks/neural-networks-first-principles`
- `07-deep-learning-systems/training-deep-networks`
- `10-transformers-llms`

## Core artifacts

- notes on graph representations, message passing, and spectral methods
- notebooks on message passing and node classification
- exercises on graph structure, Laplacians, and graph architectures

## Module emphasis

- graph structure as inductive bias
- local aggregation versus global attention
- spectral and spatial views of graph learning
- relational reasoning and a forward bridge to Module 16
