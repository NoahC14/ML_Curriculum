## Status

Explain convolutional structure and connect it to modern vision pipelines.

## Planned focus

- pooling and hierarchy

- transfer learning
- vision tasks and evaluation

## Shared toolkit usage

Vision notebooks in this module should import `shared/src/training_diagnostics.py` for:

- loss and learning-rate traces during CNN training;
- layerwise gradient norms during deep convolutional optimization; and
- confusion matrices and activation-distribution checks for classifier behavior.
