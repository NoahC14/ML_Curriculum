# Optimization Notebooks

This directory contains the interactive lab notebooks for Module 01 optimization.

Current labs:
- `OPT-01-gradient-descent.ipynb`
- `OPT-02-stochastic-and-momentum.ipynb`
- `OPT-03-second-order-methods.ipynb`
- `OPT-04-adam-and-modern-optimizers.ipynb`

Each notebook is designed to run top-to-bottom in the standard environment and uses shared helpers from `../src/optim_utils.py`.
