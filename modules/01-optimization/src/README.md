# Optimization Source Utilities

This directory contains reusable support code for Module 01 optimization labs.

Current contents:
- `optim_utils.py`: controlled 2D objectives, optimizer implementations, convergence summaries, and plotting helpers shared by the optimization notebooks.

Design intent:
- keep reusable numerical logic out of notebook cells;
- keep implementations small enough for students to inspect directly; and
- support clean top-to-bottom execution in the standard course environment.
